Основная идея - Трекер задач с совместным доступом

Функция проекта:
    Дать возможность человеку
    1. Создавать совместные задачи с другими людьми
    2. Отслеживать задачи
    3. Изменять задачи и их состояние, удалять
    4. Добавление друзей. Только с друзьями можно создавать совместные задачи. Это сделано для того, чтобы рандомные 
    люди не могли ставить задачи, которые будут связаны с тобой
    5. Смотреть задачи других людей на их страницах

Возможности связаные с Постами ( задачами ):
    1. Создание названия, описания, визуального оформления, дополнительной информации в виде видео, картинок и тд (файлов),
    Установка дедлайнов и приоритета задачи, ответственных за задачу.
    2. Просмотр задач в виде календаря, на котором они будут отображаться
    3. Возможность поделиться задачей с другими людьми ( просто для просмотра. Как пример: пост в вк )
    4. Добавление других людей к обсуждению, чтобы они могли отмечать выполненые задачи, или комментировать их
    5. Возможность скрыть задачу от просмотра другими людьми , которые не связаны с обсуждением

Стоит уточнить, что пока что в моей голове создание задачи выглядит так, что один человек создает обсуждение, в которое
он может добавить других людей. Все эти люди будут видеть все задачи, которые появляются в этом обсуждении.

База данных для прокта - PostgreSQL
